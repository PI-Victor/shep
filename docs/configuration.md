Configuration
---
By running `shep config`, the application will generate a default configuration in the current working directory.  
e.g.
```json
{
 "debugLevel": 4,
 "ircServers": null,
 "gitHub": null,
 "jenkinsServers": null,
 "travisToken": "",
 "redis": {}
}
```

`debugLevel` - The application log verbosity.   
4 - Debug  
3 - Info  
2 - Warn  
1 - Error
